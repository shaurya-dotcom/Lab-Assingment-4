# Web Dev I – Lab Assignment 4  
## Prompt Quizzer – A Quiz Game

This project is a small JavaScript quiz game created for **Web Dev I Lab Assignment 4**.  
The quiz runs completely in the browser using `prompt()` and `alert()` for input and output.

---

## Features

- Predefined list of quiz questions and answers stored in an array
- Uses a `for` loop to go through each question one by one
- Takes user input with `prompt()`
- Shows immediate feedback for each answer using `alert()`
- Normalizes input using `toLowerCase()` and `trim()` for accurate comparison
- Tracks the user’s score and displays the final result at the end
